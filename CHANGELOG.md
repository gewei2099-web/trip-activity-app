# 更新记录

格式：`## [日期] 标题` +  bullet 列表。每次功能性修改后，在**本文件最顶部**（此段说明下方）添加一条记录。

---

## [2025-03-06] 活动地图「全部」视图全局序号

- **选「全部」时**：标记显示跨天的全局顺序（1、2、3…），按日期+时间排序
- **选某天时**：保持原有当日序号（1、2、3…）
- **颜色不变**：仍按日期分色，便于区分不同天的活动

## [2025-03-05] 导出文件名增强

- **导出 JSON**：文件名增加时分秒，格式 `trip-activity-backup-YYYY-MM-DD-HHmmss.json`，同一天多次导出可区分

## [2025-03-05] 地图瓦片扩展与默认切换

- **默认改为 OSM**：地图瓦片默认使用 OSM 标准样式
- **新增可选瓦片**：Carto 浅色/深色、OpenTopoMap 地形图、CyclOSM
- **下拉显示中文名**：OSM 标准、Carto 浅色、Carto 深色、地形图、CyclOSM

## [2025-03-05] 行程活动地图增强

- **活动编号**：按时间排序后自动编号，标记上显示当日顺序（1、2、3…）
- **按日期分色**：不同日期使用不同颜色标记
- **Popup 增强**：显示日期、序号、类型、费用、首张图片
- **同日连线**：同一天的活动用彩色折线连接
- **箭头方向**：连线末端显示箭头，表示行进方向
- **按日期筛选**：多天行程时可筛选「全部」或指定日期
- **点击联动**：点击标记后下方日程列表滚动到对应活动
- **地图样式切换**：浅色 / OSM 瓦片切换
- **全屏**：地图右上角全屏按钮

## [2025-03-05] 高德安全密钥与测试连接

- **高德安全密钥**：设置中新增「高德安全密钥」输入框，若 Key 在控制台开启了数字签名则必填
- **数字签名**：按高德规范对 address、key、output 排序拼接后加私钥做 MD5，生成 sig 参数
- **测试地点搜索**：设置页增加「测试地点搜索」按钮，使用当前配置搜索「北京」以验证 Key 与网络

## [2025-03-05] 地点搜索：环境切换 + 高德 + Geoapify

- **环境切换**：设置中可选「自动 / 国内优先 / 海外优先」，自动模式通过 ipinfo.io 检测网络区域
- **高德 Key**：国内环境优先使用高德地理编码（约 6000 次/天免费）
- **Geoapify Key**：海外环境优先使用 Geoapify（约 3000 次/天免费）
- **调用链**：国内链 = 高德 → Nominatim → Photon → Open-Meteo；海外链 = Geoapify → Nominatim → Photon → Open-Meteo
- **导入/导出**：地点搜索配置一并导出与导入

## [2025-03-05] 地点搜索增强：英文回退与多数据源

- **英文回退**：中文地点（如「新加坡樟宜机场」）在 Nominatim/Photon 无结果时，自动尝试英文名（Changi Airport Singapore）再查
- **Open-Meteo**：新增第三地理编码数据源
- **常见机场**：预置北京首都、浦东、虹桥、樟宜等机场的英文回退

## [2025-03-05] 活动地图与自动地理编码

- **经纬度标签修正**：TripForm、ActivityForm 中「经度/纬度」标签与 lat/lng 正确对应
- **自动地理编码**：保存行程/活动时，若填写了地点但无经纬度，自动调用地理编码填充坐标
- **TripDetail 活动地图**：行程详情页嵌入小地图，展示当前行程内所有带坐标的活动；无坐标时显示引导文案与「去编辑」链接
- **选点状态合并**：选地点、地图选点后使用单次 setState 更新，避免多次更新导致数据丢失

## [2025-03-03] 行程日期选择器（年/月/日下拉）

- **DateSelect**：因原生 `type="date"` 在部分手机上无确定按钮无法确认，改用年/月/日下拉选择，选中即生效
- 修复新建行程时无法选择日期：仅选年份时先填充 01-01，支持从空白逐步选择年→月→日
- 行程开始/结束日期、单独活动日期均改用 DateSelect

## [2025-03-03] 地图选点增强

- **当前位置**：打开地图选点时显示蓝色圆点标识当前位置（需授权定位）
- **已选点**：若已通过文字搜索选取地点，地图上预显示该点及地名，可直接确认

## [2025-03-03] 中文地名与地图瓦片

- **地理编码**：Nominatim / Photon / 反向地理编码 增加 `accept-language: zh-CN,zh`，国内外地点均优先显示中文
- **地图瓦片**：默认改用 Carto Light（全球 CDN，国内可访问性更好），OSM 官方瓦片在国内可能无法加载
- **Leaflet CSS**：改为从 cdnjs (Cloudflare) 加载，提升国内加载成功率

## [2025-03-03] 地图选点

- **MapPicker**：新增地图选点弹窗，在地图上点击即可选择经纬度，无需手动输入
- **反向地理编码**：选点后自动调用 Nominatim 获取地点名，国内外均可用（国外反向解析更准）
- **TripForm / ActivityForm**：地点行增加「地图选点」按钮，搜不到地点时可在地图上直接选点
- 地点搜索失败时的提示改为「请点击『地图选点』在地图上选位置」

## [2025-03-03] GitHub Pages 工作流修复

- 支持 `main` 与 `master` 分支
