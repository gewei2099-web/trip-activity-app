# 行程活动地图增强想法

以下为可选增强方案，附可行度（★ 越多越易实现）与简要说明，供选择参考。

---

## 一、标记与信息展示

| 想法 | 可行度 | 说明 |
|------|--------|------|
| **活动编号** | ★★★★★ | 在标记上显示 1、2、3… 表示当日顺序。使用 Leaflet DivIcon 绘制圆形+数字，无需额外依赖。 |
| **按日期分色** | ★★★★★ | 不同日期用不同颜色标记（如第 1 天蓝、第 2 天绿）。可复用 DivIcon，数据已有 day.date。 |
| **按类型分图标** | ★★★★★ | 景点、餐厅、交通、住宿用不同图标/颜色。准备 5 种小图标或 CSS 样式即可。 |
| **缩放时显示日期** | ★★★★☆ | 缩放级别 &lt; 12 时只显示点；≥ 12 时在标记旁显示日期。用 `useMapEvents` 获取 zoom 控制 Popup 内容。 |
| **Popup 内容增强** | ★★★★★ | 在 Popup 中增加：日期、类型标签、费用、缩略图。数据已有，仅改 JSX。 |

---

## 二、路线与顺序

| 想法 | 可行度 | 说明 |
|------|--------|------|
| **连线（按日）** | ★★★★☆ | 用 Leaflet Polyline 连接同一天的活动，形成当日路线。需在 collectMarkers 中保留 day 信息。 |
| **箭头方向** | ★★★★☆ | 在连线上加箭头表示顺序。可用 Polyline 的 `arrowheads` 或装饰线，或 `leaflet-ant-path` 等。 |
| **序号 + 日期组合** | ★★★★★ | 如 `3·3/15`：第 3 天第 3 个活动，3 月 15 日。DivIcon 内写两行文字即可。 |

---

## 三、交互与筛选

| 想法 | 可行度 | 说明 |
|------|--------|------|
| **按日期筛选** | ★★★★★ | 地图上方加日期 Tab 或下拉，选中某天只显示该天活动并 fitBounds。 |
| **按类型筛选** | ★★★★☆ | 勾选类型（景点/餐厅等），地图仅显示该类型活动。需增加筛选 state。 |
| **点击标记滚动到日程** | ★★★★☆ | 点击地图标记时，下方日程列表滚动到对应活动。可用 ref + scrollIntoView。 |
| **侧边活动列表** | ★★★★☆ | 地图旁显示可点击的活动列表，点击后地图定位并弹出 Popup。需调整布局。 |

---

## 四、视觉与样式

| 想法 | 可行度 | 说明 |
|------|--------|------|
| **地图样式切换** | ★★★★☆ | 已有 mapTiles（Carto/OSM），加一个切换按钮即可。 |
| **全屏地图** | ★★★★☆ | 右上角加全屏按钮，使用 `map.requestFullscreen()` 或全屏容器。 |
| **标记聚合** | ★★★☆☆ | 点多时缩略图显示聚合，放大后拆分。需 `react-leaflet-cluster` 或类似库。 |
| **热力效果** | ★★☆☆☆ | 用热力图表示活动密集区。需 `leaflet.heat`，更适合大量点，行程场景意义有限。 |

---

## 五、数据与计算

| 想法 | 可行度 | 说明 |
|------|--------|------|
| **点间距离** | ★★★★☆ | 在 Popup 或连线旁显示「距上一站约 X km」。用 Haversine 公式，无外部 API。 |
| **路线动画** | ★★☆☆☆ | 沿连线播放移动动画。实现复杂，且需考虑跨日、折返等情况。 |
| **预计时长** | ★★☆☆☆ | 基于距离估算步行/车行时长。需路由 API 或简化公式，准确度一般。 |

---

## 六、推荐组合（按优先级）

1. **基础增强（易做）**：活动编号 + Popup 增强 + 按日期分色  
2. **路线感**：同一天连线 + 箭头 + 按日期筛选  
3. **交互**：按日期筛选 + 点击标记滚动到日程  
4. **视觉**：地图样式切换 + 全屏  

---

## 七、需补充的数据

当前 `collectTripMarkers` 未携带 `dayDate`、`dayIndex`、`actIndex`，部分想法需扩展为：

```javascript
list.push({
  ...a,
  dayDate: d.date,
  dayIndex: dayIdx,
  actIndex: actIdx,
  sequence: /* 当日第几个 */
})
```

选好要做的增强后，可据此调整数据与 UI。
